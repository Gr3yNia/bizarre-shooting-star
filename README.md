# Bizarre Adventure of a Shooting Star  
### Expressive Wearable Project

This project explores the idea of the human body as a moving system, like a shooting star.

Using servo motors and a light sensor, the wearable creates continuous motion that responds to the environment. The movement is subtle and layered, representing internal activity that is always happening, even when it is not visible.

## Components
- Adafruit Circuit Playground  
- Servo motors (360° + 180°)  
- Light sensor  
- Aluminum sculpting wire  
- Wool (crochet)  
- Translucent plastic
